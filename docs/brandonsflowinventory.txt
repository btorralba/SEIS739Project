User -> UI: navigate to admin site,\nenter creds

UI -> Backend: getProducts()\ngetOrders()\ngetCustomers()

Backend --> UI: product objects\norder objects\ncustomer objects
UI -> UI: sets datasource for table
UI --> User: renders admin view
User -> UI: edits a field
UI -> Backend: updateOrder(orderInfo)\nupdateProduct(productInfo)
Backend -> DB: update ...
DB --> Backend: success
Backend --> UI: success
UI --> User: renders updated view
